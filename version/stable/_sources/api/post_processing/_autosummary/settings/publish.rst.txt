This is most important when generating reports in batch mode where the user may only enter an output directory without specifying a name.

:ref:`generate_3d_viewer_files <post_processing_generate_3d_viewer_files>`               Indicates whether to generate ANSYS Viewer Files for figures that are included in a report. When generating report, an AVZ file for each figure will be generated. Both AVZ and ARZ files can be displayed in stand alone ANSYS Viewer. It should be noted that when generating HTML reports that the ANSYS Viewer Files cannot be directly accessed from the report when viewed in a standard Web Browser.
